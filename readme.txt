Aakaari - Minimal WooCommerce Theme
----------------------------------
Install:
1. Upload aakaari-theme.zip via Appearance -> Themes -> Add New -> Upload Theme
2. Activate Aakaari.
3. Install & activate WooCommerce plugin.
4. Set up menus and customize the look via Appearance -> Customize.

Notes:
- This is a starter theme. Expand CSS under style.css or add assets/css/ and adjust templates as needed.
- To customize the WooCommerce templates, copy the files from the woocommerce/ folder into the child theme or modify here.
